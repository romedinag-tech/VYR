# Fotografías por proyecto — cartera VyR

Una imagen **ad-hoc por proyecto** del listado de experiencia (Acreditación MOP,
*Certificados de experiencia con desglose*). Nombradas `NN-slug.jpg`.

> ⚠️ **Placeholders temáticos** (de Unsplash, libres de uso), elegidos por **categoría**
> de proyecto, no son fotos reales de cada obra. Pensados para reemplazar por
> fotografía propia obtenida desde los estudios. El detalle de cada archivo,
> su categoría y la fuente está en [`_indice.csv`](_indice.csv).

## Criterio de asignación por categoría
| Categoría | Tipo de imagen |
|---|---|
| Ciclovías | ciclista / ciclovía |
| Transporte público (PTU) · Terminal rodoviario | bus / transporte urbano |
| Ferroviario | tren / vía férrea |
| Vialidad · interurbano · concesión · puente | intercambio vial aéreo / ruta |
| Eje urbano · accesibilidad | avenida urbana |
| Gestión de tránsito · SCAT | calle urbana con tránsito |
| Costanera · teleférico (cerros+mar) | borde costero aéreo |

## Cómo reemplazar por la foto real
Sustituye el archivo manteniendo el **mismo nombre** (`NN-slug.jpg`) y quedará
enganchado donde se use en el sitio. Ideal: horizontal, ≥1400 px de ancho.
